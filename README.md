# WallPaper App

### It has many screens like below :
###### - AllWallPaperPage
###### - DetailsPage
###### - FavoritePage

### List of Functionalities
###### - Display all wallpapers in AllWallPaperPage
###### - Show category wise wallpapers in AllWallPaperPage
###### - Display a tapped wallpaper in DetailPage
###### - Store a wallpaper which you added to favorites
###### - Display stored/added wallpapers in favoritePage
